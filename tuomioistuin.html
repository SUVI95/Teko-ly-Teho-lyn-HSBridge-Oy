Vaihe 03

Julkaise ja visualisoi

Kaksi työkalua, kaksi eri tehtävää. Molemmat vaaditaan. Tee ne rinnakkain tai peräkkäin — järjestys on sinun.

Track A — Gamma
🎯 Gamma — Argumenttidokumentti

Rakenna Clauden argumenteista julkaisukelpoinen dokumentti Gammassa. Ei esitys — dokumentti. Yksi sivu. Sellainen joka voitaisiin oikeasti lähettää päätöksentekijälle.

Valitse ensin tapaus.
  1. Avaa Gamma → Luo uusi → Document (ei Presentation)
  2. Liitä tämä prompt
  3. Täytä hakasulkuihin tulevat kohdat omilla tiedoillasi
  4. Kirjoita OMA KANTA -osioon oma päätöksesi perusteluineen
  5. Vie tai jaa dokumentti — kopioi julkinen linkki
Avaa Gamma →
Track B — Canva
🎨 Canva — Tuomiokortti

Suunnittele A5-kokoinen visuaalinen tuomiokortti joka kertoo yhdellä silmäyksellä: tapaus, sinun kantasi, vahvin argumenttisi. Ei pitkää tekstiä — visuaalinen päätös.

Luo Canvassa A5-dokumentti (148×210 mm). SISÄLTÖ — tasan nämä elementit, ei enempää: 1. Tapauksen nimi isolla (esim. REKRYTOINTI) 2. Sinun kantasi kahdella sanalla (esim. KIELLETÄÄN / SALLITAAN / SÄÄNNELLÄÄN) 3. Vahvin argumenttisi — yksi lause 4. Yksi numero tai tilasto Perplexityn löydöistä 5. Päivämäärä TYYLI: - Vain kaksi väriä + musta/valkoinen - Ei stockkuvia, ei ikoneja - Typografia tekee kaiken työn - Näyttää viralliselta asiakirjalta, ei some-postaukselta Tallenna PNG-muodossa.
  1. Avaa Canva → Luo uusi → Custom size → 148 × 210 mm
  2. Rakenna kortti yllä olevan ohjeistuksen mukaan
  3. Lataa PNG-muodossa (Share → Download → PNG)
