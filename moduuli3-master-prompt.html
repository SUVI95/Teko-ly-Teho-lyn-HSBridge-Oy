Ymmärrys

Mitä prompt tarkoittaa?

Prompt on pyyntö tai ohje, jonka annat tekoälylle. Se kertoo tekoälylle mitä haluat — mutta myös miten haluat sen.

Tekoäly ei ajattele kuten ihminen. Se ei tiedä kontekstia, ellei kerro sitä. Se ei arvaa mitä tarkoitat, ellei selitä sitä. Prompt on sinun tapa kommunikoida tekoälyn kanssa.

"Hyvä prompt on kuin hyvä ohje — selkeä, tarkka, ja kertoo kaiken mitä tarvitaan."

Ero on selvä

Heikko vai vahva prompt?

Katso konkreettinen ero. Sama tarkoitus — täysin eri vastaukset.

❌ Heikko prompt
"Autatko minua?"
Vastaus: Yleisiä vinkkejä. Ei kontekstia, ei selkeyttä. Sopii kenelle tahansa — eli ei oikeastaan kenellekään.
✓ Vahva prompt
"Olet kokemusasiantuntija. Minulla on ongelma X tilanteessa Y. Tarvitsen kolme konkreettista vaihetta, miten ratkaisen tämän. Vastaa lyhyesti ja käytännönläheisesti."
Vastaus: Tarkka, konkreettinen ja toimiva. Tekoäly tietää roolinsa, kontekstin, tehtävän ja halutun tuloksen.

"Epäselvä prompt → Epäselvä vastaus. Tarkka prompt → Tarkka vastaus."

Rakenne

Miten hyvä prompt rakennetaan?

Hyvä prompt on kuin hyvä ohje — se kertoo kaiken mitä tarvitaan selkeästi ja järjestyksessä.

1

Anna rooli

Kerro tekoälylle kuka se on. Tämä muuttaa sen näkökulman ja vastaustyylin täysin.

"Olet kokemusasiantuntija..."
"Olet yksinkertaistaja..."
"Olet luova kirjoittaja..."
2

Anna konteksti

Kerro tilanne, tausta, tai mitä sinun pitää tietää. Mitä enemmän kontekstia, sitä parempi vastaus.

"Minulla on ongelma X..."
"Tilanteessa jossa Y..."
"Kun Z tapahtuu..."
3

Määritä tehtävä ja formaatti

Kerro tarkalleen mitä haluat ja miten haluat sen. Luettelo? Vaiheet? Yhteenveto? Lyhyt vai pitkä?

"Anna kolme konkreettista vaihetta..."
"Kirjoita lyhyt yhteenveto..."
"Listaa viisi esimerkkiä..."
