Vaihe 04

Dokumentoi ja julkaise

Kaksi työkalua, kaksi eri tehtävää. Molemmat vaaditaan ennen julkaisua.

Track A — Gamma
📄 Gamma — Työkalun dokumentaatio

Rakenna Gammassa yksi sivu joka dokumentoi työkalusi niin selkeästi että kollegasi pystyy käyttämään sitä ilman sinun apuasi.

Create a one-page professional tool documentation in Finnish. TOOL NAME: [they fill in] PROBLEM IT SOLVES: [one sentence, specific] WHO IT IS FOR: [exact user description] HOW TO USE IT — 3 steps: 1. [what to prepare] 2. [what to paste] 3. [what you get back] SYSTEM PROMPT: [paste their final system prompt here in a code block] WHAT WORKS WELL: [2 bullet points from testing] KNOWN LIMITATIONS: [2 bullet points from testing] Style: technical documentation, not a pitch deck. Clean, functional, honest about limitations. Someone reading this should be able to use the tool in 5 minutes.
  1. Avaa Gamma → Luo uusi → Document
  2. Liitä tämä prompt ja täytä hakasulut
  3. Kopioi system promptisi vaiheesta 2 mukaan
  4. Vie tai jaa — kopioi julkinen linkki
Avaa Gamma →
Track B — Canva
🎨 Canva — Työkalukortti

Suunnittele A6-kokoinen kortti joka esittelee työkalusi yhdellä silmäyksellä. Näyttää siltä kuin se olisi oikea tuote.

Luo Canvassa A6-dokumentti (105×148 mm). SISÄLTÖ — tasan nämä elementit: 1. Työkalun nimi — iso, selkeä 2. Yksi lause mitä se tekee 3. Kolme bullet-pointia: mitä saat syötettyä → mitä saat takaisin 4. Yksi lause siitä mitä työkalu EI tee (rajoite — tekee kortista uskottavan) 5. "Versio 1.0" + päivämäärä pienellä TYYLI: - Yksi pääväri + musta/valkoinen - Näyttää ohjelmistotuotteen kortilta - Ei kuvituskuvia - Kompakti ja informatiivinen Tallenna PNG-muodossa.
  1. Avaa Canva → Custom size → 105 × 148 mm
  2. Rakenna kortti ohjeen mukaan
  3. Lataa PNG (Share → Download → PNG)
