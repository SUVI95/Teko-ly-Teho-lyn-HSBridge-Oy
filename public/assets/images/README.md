# Logo Files

Place your logo files in this folder.

## Supported Formats
- PNG (.png)
- JPEG (.jpg, .jpeg)

## File Names
The system will look for:
- `logo.png` (preferred)
- `logo.jpg` or `logo.jpeg` (alternative)

## Usage
After adding your logo file, make sure it's named `logo.png` (or update the references in the HTML files if using a different name).

## Current Logo
The logo is used in:
- Front page (index.html) navigation
- Login page header
- Homework page navigation
